1. Такого отображения можно добиться, используя метатег viewport, со следующими свойствами content="width=device-width, initial-scale=1.0", ширина страницы сайта -по ширине девайса, initial-scale=1, что размер страницы будет 100% величины области просмотра.
2. Отзывчивый дизайн оставляет расположение элементов в том же виде, не меняя положение блоков, просто подгоняя все элементы под размер экрана за счет заданных относительных единиц измерения. Адаптивный дизайн включает в том числе изменение положения блоков, отключение каких-либо функций, присваивание блокам другой структуры например, было две колонки рядрм, стала одна колонка один блок под другим. 
3. Лучше использовать относительные единицы: em, rem.
4. На первой картинке отзывчивая верстка, на второй-адаптивная. Не совсем понятен вопрос к какой категории шаблонов относится, к категории новостных сайтов наверное. Блочная, если это категория.
5. Стили  для экранов шириной от 800 до 1200 пикселей можно задать с помощью медиа-запроса @media screen and (min-width: 800px) and (max-width: 1200px)
6. Внутри таблицы css через запрос @media screen and (min-width:...) and (max-width: ...), в коде страницы внутри тега style, через тег link в коде html, через запрос @import.
7. Свойство max-width: 100% позволяет делать картинки гибкими.таким образом картинка будет уменьшаться в зависимости от размера экрана, но никогла не будет увеличиваться до размеров, превышающих ее размер. 
8. Landscape -это поворот экрана горизонтально, portrait -вертикально, можно задать стили, который будут применяться только при условии поворота экрана landscape , то есть горизонтально, или создать медиа-запрос со стилями только для режима portrait, то есть вертикального положения экрана. Создается @media (orientation: landscape).
9. Через dev tools в настройках браузера,  через физическое использование разных девайсов, через онлайн-сервисы, предназначенные для тестирования разных размеров браузера. 
10. Через атрибут srcset, к котором сразу указываются размеры изображений.
